PokemonConquest Like

Description

Notre Projet est une adaptation sur ordinateur en langage JAVA du jeu Pokémon Conquest, inspiré de la célèbre série de jeux vidéo Fire Emblem. Le jeu est jouable à 2 joueurs avec chacun sa propre fenêtre de jeu.
